Overview

The Hive is a strategy game on Steam that shipped before I joined the team. I worked on post-release patching/hotfixes, save/load reliability, and DLC feature work (including a survival-focused mode with persistent units and equipment).

Inventory/equipment UI + tutorial flow.

Tooltip formatting and stat display.

Control groups and selection UI improvements.

Unity C# Live Ops Save/Load UI Gameplay

What I shipped / fixed

Autosave & save/load reliability

  • Implemented autosave with anti-spam and cutscene safety checks
  • Integrated autosave triggers into cutscenes and objective completion
  • Fixed save/load issues (e.g., starter units not saving, disappearing units, difficulty not restoring correctly)
  • Updated save/load UI to clearly separate manual saves, quick saves, and auto saves

DLC Survival mode persistence

  • Implemented DLC Survival mode features where units persist between levels
  • Persisted unit equipment and inventory state across levels and sessions
  • Ensured mode rules integrate cleanly with existing save/load flows

Unit selection + control groups UI

  • Added Group Selection UI and improved control group HUD behavior
  • Fixed edge cases in rapid double-click selection (unit isolation vs group selection)
  • Improved group counts and live updates (including resurrected units)
  • Added inventory button pulse when new items are received

Tooltip system + UI clarity

  • Implemented unit tooltip templates and formatting (token replacement like {DMG}, {HP}, {DEF}, etc.)
  • Exposed attack/defence tooltip messages on units and wired into HUD
  • Improved tooltip positioning: pivot adjustment + clamping within UI bounds
  • Added tooltip messages across inventory/resource UI for better player guidance

UI prefabs and resource popups

  • Added Resource Popup UI and supporting scripts (pickup feedback)
  • Updated Item/Resource UI to include additional resource icons
  • Normalized prefab serialization and updated renderer/particle settings across assets

Tutorial system improvements

  • Introduced TutorialManager prefab and updated scene integration
  • Added multi-page tutorial panels with keyboard navigation
  • Added runtime tutorial reset (command line flag + hotkey) for testing

Tools and tech

  • Engine: Unity
  • Language: C#
  • Work type: live patching/hotfixes, DLC development, UI systems, persistence fixes
